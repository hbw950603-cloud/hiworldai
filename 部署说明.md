## 🚀 部署步骤

### 方法 1: Vercel Dashboard 部署（推荐）

1. **访问** https://vercel.com
2. **登录** 你的账号
3. **删除旧项目**（如果有）
   - 找到 hiworldai 项目
   - Settings → Delete Project
4. **创建新项目**
   - 点击 "Add New Project"
   - 选择 "Deploy a folder"
   - 选择 `/home/zhenyi/hiworldai/` 文件夹
5. **等待部署完成**（约 1 分钟）
6. **绑定域名**
   - Settings → Domains
   - 添加 hiworldai.com
   - 配置 DNS（见下方）

### 方法 2: Vercel CLI 部署

```bash
cd /home/zhenyi/hiworldai
vercel --prod
```
