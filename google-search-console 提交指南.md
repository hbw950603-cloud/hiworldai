# Google Search Console 提交指南

## 网站信息
- **网站**: https://hiworldai.com
- **Sitemap**: https://hiworldai.com/sitemap.xml
- **Google Analytics**: G-JRCSBY0SLM

## 提交步骤

### 1. 访问 Google Search Console
https://search.google.com/search-console

### 2. 添加属性
- 选择"网址前缀"
- 输入：https://hiworldai.com
- 点击"继续"

### 3. 验证所有权

**方法 A: HTML 文件上传（推荐）**
1. 下载验证 HTML 文件
2. 上传到网站根目录
3. 点击"验证"

**方法 B: HTML 标签**
1. 复制 meta 标签
2. 添加到网站首页 `<head>` 部分
3. 点击"验证"

**方法 C: DNS 记录**
1. 添加 TXT 记录到域名 DNS
2. 等待 DNS 传播
3. 点击"验证"

### 4. 提交 Sitemap
1. 左侧菜单点击"站点地图"
2. 输入：sitemap.xml
3. 点击"提交"

### 5. 等待收录
- 通常 1-7 天内开始收录
- 可在"覆盖率"报告查看状态

## 追踪指标

| 指标 | 目标（30 天） |
|------|--------------|
| 收录页面 | 20+ |
| 搜索展示 | 1000+ |
| 搜索点击 | 100+ |
| 平均排名 | 前 20 |

## 优化建议

1. **提交后每天检查** - 查看覆盖率报告
2. **修复错误** - 404、服务器错误等
3. **监控搜索词** - 了解用户如何找到你
4. **优化低表现页面** - 提升排名
